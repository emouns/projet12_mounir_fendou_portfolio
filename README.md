# Portfolio — Mounir Fendou
> Intégrateur Web | HTML · CSS · JavaScript

## Lien en ligne
https://mounir-fendou-portfolio.netlify.app

## Stack
- HTML5 sémantique
- CSS3 (Flexbox, Grid, animations, variables CSS)
- JavaScript ES6+ (IntersectionObserver, DOM, validation)
- Font Awesome 6 (icônes)
- Google Fonts (Playfair Display + Inter)

## Fonctionnalités
- Splash screen animé au chargement
- Navigation sticky avec hover animé
- Menu hamburger responsive
- Scroll reveal (IntersectionObserver)
- Formulaire de contact validé en JavaScript
- Responsive : desktop, tablette, mobile
- Accessible : aria, focus-visible, prefers-reduced-motion
- SEO : title, meta description, Open Graph

## Méthode de développement
Ce portfolio est codé à la main de A à Z (HTML/CSS/JS), 
sans framework ni générateur de site.


## Structure 

- index.html
- css/style.css
- js/main.js
- assets/
- README.md
