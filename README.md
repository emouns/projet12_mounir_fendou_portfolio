##  Fonctionnalités

- **Splash screen** — écran de chargement animé au lancement (2 secondes)
- **Navigation sticky** — barre de navigation fixe avec lien actif au scroll
- **Menu hamburger** — menu responsive pour mobile avec accessibilité clavier
- **Scroll reveal** — apparition progressive des sections au défilement
- **Formulaire de contact** — avec validation JavaScript et messages d'erreur accessibles
- **Responsive** — adapté mobile, tablette et desktop

---

##  Accessibilité & SEO

- Balises HTML sémantiques (`<main>`, `<nav>`, `<section>`, `<article>`)
- Attributs `aria-label`, `aria-expanded`, `role` sur les éléments interactifs
- Navigation au clavier complète avec focus visible
- Balise `<title>` et `<meta name="description">` pour le SEO
- Hiérarchie des titres respectée (H1 → H2 → H3 → H4)
- `prefers-reduced-motion` respecté
